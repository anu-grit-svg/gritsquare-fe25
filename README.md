# Gritsquare

Gritsquare is a public message board website created for the Version Control course at Grit Academy.

## Project summary

The website allows users to write a message and see it displayed on the page.  
The messages are saved using Firebase Realtime Database.

## Pages

- Home
- About
- Contact

## Features

- Public message board
- Users can write messages
- Messages are displayed on the page
- Messages are saved using Firebase
- Responsive design
- Hosted with GitHub Pages

## Technologies used

- HTML
- CSS
- JavaScript
- Firebase Realtime Database
- Git
- GitHub
- GitHub Pages
